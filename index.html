Design Summary

For this project I was looking to make use of the drawing tablet screen I had recently obtained. I went to trusty ole sharpen to grab a prompt, and I pulled “Design a sticker for a bakery in Lisbon”. With the new drawing tablet I decided to switch up my sketching game and use something digital, thankfully windows has an app called whiteboard which was very straightforward to use and exactly what I needed to get to sketching.

The Process

Getting started I did the tried and true listing of attributes as I have done for all my design challenges. The attributes are as follows: Capitol City, Baked Goods, Portuguese Baked Goods such as, Ovos Moles, Fidalgo, Pao de Lo, and Queijadas. Jumping off the Pao de Lo, I went with the name Pao de Lisbon, as a nod to the Portuguese sponge cake so many enjoy.

Final

For the final I really wanted to give it a lighthearted colorful feel, I used my drawing pad to hand write the text. Drew the pan de lo for the focal point of the sticker and created a nice colorful piece to complete the sticker for the bakery in Lisbon. Someday I hope to travel to Portugal to try some of these treats for myself. I really enjoyed using the drawing pad and can’t wait to get more into using it as I continue to design.
